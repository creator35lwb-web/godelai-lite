### 4.2 GodelAI-Lite Architecture

```
User Input
    ↓
[GIFP-Lite] → Identity prompt + behavioral constraints
    ↓
[MemPalace-Lite] → Episodic history + extracted facts + patterns
    ↓
[Augmented Prompt]
    ↓
[Gemma 4 Inference] → temp=0.7, top_p=0.9, max_tokens=512
    ↓
[MACP-Lite] → Reasoning chain step recorded
    ↓
[GIFP Consistency Check] → Score response (0.0–1.0)
    ↓
[Memory Update] → Store interaction + facts + behavior
    ↓
[Optional Refinement] → If consistency < 0.8 and refine=True (max 2 iterations)
    ↓
Final Response
```

### 4.3 Critical Weaknesses to Fix (Priority Order)

| # | Weakness | Fix | Impact |
|---|----------|-----|--------|
| 1 | No baseline comparison | Add side-by-side: plain Gemma 4 vs GodelAI-Lite on 5–10 standardized prompts | **Highest — judges need proof it works** |
| 2 | Naive fact extraction (keyword heuristics) | Replace with sentence-transformer similarity or spaCy NER | High — current approach will false-positive constantly |
| 3 | No disk persistence | Serialize MemPalace to JSON/SQLite; survive kernel restarts | High — table stakes for real usage |
| 4 | Shallow consistency check (4 keywords) | Use cosine similarity across response embeddings | Medium — brittle heuristic undermines GIFP claims |
| 5 | No memory decay | Add timestamp-based relevance decay to MemPalace | Medium — stale facts accumulate indefinitely |
| 6 | Weak evaluation | Multi-fact retention, multi-turn consistency, semantic coherence scores | Medium — current eval too simple |

### 4.4 Winning Narrative Strategy

The submission must communicate three things clearly:

1. **The Problem:** SLMs fail not from lack of intelligence, but from lack of memory continuity across turns. Gemma 4 alone forgets everything between calls.

2. **The Solution:** GodelAI-Lite adds a three-layer augmentation (runtime memory + reasoning continuity + identity governance) with zero fine-tuning required. Works on any SLM.

3. **The Ecosystem Connection:** This is not an isolated project. It is the inference-time companion to the training-time GodelAI framework. Together they close both memory gaps — weight-level AND context-level. This is a system, not a hack.
